# Setup 
## Download

## Controls 
[ARROW_UP] Move up <br />
[ARROW_DOWN] Move down <br />
[ARROW_LEFT] Break <br />
[ARROW_RIGHT] Accelerate <br />
[SPACE] Shoot <br />
[F5] Restart <br />
[F11] Fullscreen <br />
