## Controls 
[ARROW_UP] Move up \n\r
[ARROW_DOWN] Move down \n\r
[ARROW_LEFT] Break \n\r
[ARROW_RIGHT] Accelerate \n\r
[SPACE] Shoot \n\r
[F5] Restart \n\r
[F11] Fullscreen \n\r
